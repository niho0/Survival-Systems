# Survival-Systems

> 独立开发的生存模拟类2D游戏系统，有完整的游戏框架，代码量约2500行

## 🎥 视频演示
[(https://www.bilibili.com/video/BV18DB9BaEmF/)]

## 核心系统与技术实现
| 系统模块 | 功能亮点和技术要点 |
| :--- | :--- |
| **迷雾系统** | 实时探索揭露迷雾，像素更新，世界坐标到UV坐标转换 |
| **权重随机事件系统** | 事件的互斥和因果，事件的条件触发与条件限制 |
| **背包与资源管理系统** | 完整的UI交互，道具数量堆叠 |

## 开发版本

Unity 6.0(6000.0.60f1)
